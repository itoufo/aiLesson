# 第2章：PBLプロジェクト設計

## 学習目標

この章を終えると、以下のことができるようになります：

- PBL（課題解決型学習）の概念と効果を理解できる
- 実践的なプロジェクトテーマを設定できる
- チームでAIを活用した協働作業ができる
- プロジェクトを計画・実行・評価できる

---

## 2.1 PBL（Project-Based Learning）とは

### 2.1.1 PBLの定義

**PBL（課題解決型学習）** は、現実の課題に取り組むことで知識とスキルを習得する学習方法です。

![従来の学習 vs PBL](/docs/images/03_発展/02_traditional_vs_pbl.jpeg)

### 2.1.2 AI活用PBLの特徴

| 要素 | 内容 |
|------|------|
| **現実課題** | 企業・地域の実際の課題に取り組む |
| **AI活用** | P-A-I-Cサイクルでプロジェクトを推進 |
| **チーム協働** | 多様なバックグラウンドのメンバーと協力 |
| **成果物** | 実用可能なアウトプットを作成 |
| **振り返り** | プロセスと結果の両面を評価 |

### 2.2.2 テーマカテゴリー

| カテゴリー | テーマ例 | 対象 |
|------------|----------|------|
| **企業DX** | 業務効率化、データ活用、自動化 | 企業・組織 |

| **環境・SDGs** | 脱炭素、循環経済、社会課題 | 社会貢献 |

## 2.5 プロジェクトテーマ例

### 2.5.1 企業向けテーマ例

```markdown
【テーマ1：カスタマーサポート効率化】
課題：問い合わせ対応に時間がかかり、顧客満足度が低下
目標：AIを活用した対応効率化と満足度向上策の提案
成果物：現状分析レポート、改善提案書、プロトタイプ

【テーマ2：社内ナレッジ活用促進】
課題：社内の知見が属人化、ノウハウ共有が進まない
目標：AIを活用したナレッジマネジメント施策の立案
成果物：課題分析、ソリューション提案、運用計画

【テーマ3：営業プロセス最適化】
課題：営業活動の非効率、成約率の低迷
目標：AIを活用した営業支援策の提案
成果物：営業プロセス分析、AI活用提案、効果予測
```

### 2.5.2 地域・社会向けテーマ例

```markdown
【テーマ4：地域観光振興】
課題：観光客減少、地域の魅力が伝わっていない
目標：AIを活用した観光プロモーション戦略の策定
成果物：観光分析レポート、プロモーション企画書、コンテンツ案

【テーマ5：高齢者見守りサポート】
課題：独居高齢者の孤立、見守りの人手不足
目標：AIを活用した見守りサービスの企画
成果物：ニーズ調査レポート、サービス企画書、プロトタイプ

【テーマ6：SDGs推進プロジェクト】
課題：企業・地域のSDGs取り組みが進まない
目標：AIを活用したSDGs推進策の提案
成果物：現状分析、取り組み提案、効果測定計画
```

## 2.6 実践演習

### 演習：プロジェクト計画の策定

```
【課題】
以下のいずれかのテーマを選択し、
4週間のPBLプロジェクト計画を策定してください。

テーマ選択肢：
A) 自社（または想定企業）の業務効率化
B) 地域の課題解決（観光、福祉、教育等）
C) 新サービス・新規事業の企画

作成するもの：
1. プロジェクト計画書（テンプレート使用）
2. WBS・スケジュール
3. AI活用計画
4. チーム編成案（想定）

評価基準：
- テーマの適切性
- 計画の具体性と実現可能性
- AI活用の効果的な設計
- リスクへの考慮
```

良いプロジェクトテーマの条件として不適切なものは？

- [ ] A) 現実の課題である

## 章末まとめ

### 本章のポイント

1. **PBL**は現実課題に取り組むことで実践的スキルを習得する学習方法
2. **テーマ設定**は現実性、適切な難易度、AI活用適性が重要
3. **計画策定**ではスコープ、スケジュール、役割分担を明確に
4. **週次サイクル**で計画・実行・振り返りを繰り返す
5. **チームコラボレーション**では心理的安全性とAI活用の共有が鍵
