# 演習問題集

本書は、各段階の学習内容を定着させるための演習問題集です。
実際にAIを使って取り組んでください。

## 第1段階（入門）演習

### 演習1-1：AIリテラシー確認

**問題**：以下の文章の正誤を判定し、誤りがあれば正しい説明を書いてください。

1. 現在のAIは人間と同等の知能を持っている
2. ChatGPTは学習データ以降の情報を知らない
3. AIは常に正確な回答をする
4. 生成AIは統計的なパターンマッチングで動作する
5. AIが生成した文章には著作権が発生する

### 演習1-2：プロンプト作成

**問題**：以下の目的に合ったプロンプトを作成してください。

**シナリオ1**：
新入社員向けのビジネスマナー研修資料を作成したい。
挨拶、名刺交換、電話応対の3つをカバーし、
各項目5分程度で説明できる分量にしたい。

**シナリオ2**：
顧客からのクレームメールへの返信文を作成したい。
商品に傷があったという内容で、謝罪と対応策を伝える必要がある。
顧客との関係を損なわない丁寧な文面にしたい。

**シナリオ3**：

10分程度の発表時間を想定。

### 演習1-3：要約練習

**問題**：以下の文章を、指定された条件で要約してください。

**元の文章**：
```
デジタルトランスフォーメーション（DX）は、企業がデジタル技術を
活用して、ビジネスモデルや業務プロセス、組織文化を変革し、
競争上の優位性を確立することを指します。単なるIT化やデジタル化
とは異なり、DXは企業の根本的な変革を伴います。

日本企業のDX推進には多くの課題があります。レガシーシステムの
刷新、デジタル人材の不足、経営層のコミットメント不足、
部門間の縦割り意識などが障壁となっています。経済産業省は
「2025年の崖」として、DXが進まない場合の経済損失を
年間最大12兆円と試算しています。

DXを成功させるためには、経営者のリーダーシップ、明確なビジョン、
組織全体での取り組み、そして段階的なアプローチが重要です。
また、外部パートナーとの連携や、失敗を許容する文化の醸成も
成功の鍵となります。
```

**要約条件**：
- 条件A：50文字以内の一文で要約
- 条件B：箇条書き3点で要約
- 条件C：経営者向けに100文字で要約

## 第2段階（応用）演習

### 演習2-1：Chain-of-Thought実践

**問題**：以下の問題をCoTで解いてください。

**ビジネス計算問題**：
```
ある店舗の月間データ：
- 来店客数：5,000人
- 購入率：30%
- 平均客単価：3,000円
- 粗利率：40%
- 固定費：200万円

来月から以下の施策を実施予定：
- Web広告で来店客数を20%増加させる（広告費50万円/月）
- 会員限定セールで購入率を5%向上させる
- まとめ買い促進で客単価を10%向上させる

この施策により、来月の営業利益はいくらになるか？
施策を行わなかった場合と比較して、投資対効果は妥当か？
```

### 演習2-2：情報整理（MECE・ピラミッド）

**問題**：以下のデータをMECEに分類し、ピラミッド構造で整理してください。

**データ**：社員アンケート結果（複数回答可）
```
「職場環境で改善してほしい点」

・残業が多い（45%）
・給与が低い（38%）
・有給が取りにくい（32%）
・オフィスが狭い（28%）
・人間関係が悪い（25%）
・評価制度が不透明（23%）
・研修が少ない（20%）
・リモートワークができない（18%）
・福利厚生が少ない（15%）
・通勤が大変（12%）
・設備が古い（10%）
```

**タスク**：
1. 上記をMECEなカテゴリに分類
2. 経営層への報告用にピラミッド構造で整理
3. 優先的に取り組むべき課題を3つ提言

### 演習2-4：業務自動化設計

**問題**：以下の業務を自動化するフローを設計してください。

**現状の業務**：
```
日次売上報告の作成

1. 各店舗からの売上報告メールを確認（10店舗）
2. メールの添付ファイル（Excel）を開く
3. 各店舗の売上をまとめシートに転記
4. 前日比、前年同日比を計算
5. グラフを更新
6. 異常値（前日比±20%以上）があれば赤字でマーク
7. PDFに変換
8. 関係者（5名）にメールで送付
9. 共有フォルダに保存

所要時間：約90分/日
課題：手作業でミスが発生、担当者休暇時に対応できない
```

**作成するもの**：
1. 自動化フロー図
2. 使用するツール/サービスの提案
3. 想定される効果（時間削減、エラー削減）
4. 導入ステップ

## 第3段階（発展）演習

### 演習3-1：P-A-I-Cサイクル実践

**問題**：以下のテーマでP-A-I-Cサイクルを実践してください。

**テーマ**：「社内研修プログラムのAI活用提案」

**背景**：
あなたの会社では、年間100回以上の社内研修を実施しています。
研修の企画、資料作成、運営、効果測定に多くの工数がかかっています。
AIを活用して研修プログラムを効率化・高度化する提案を求められました。

**実践内容**：
1. Prompt：初期の問いかけを作成
2. Ask：AIとの対話で深掘り（5回以上）
3. Iterate：出力を評価しフィードバック（3回以上）
4. Create：最終提案書を作成

### 演習3-4：AI倫理ディスカッション

**問題**：以下のケースについて、倫理的観点から分析し、
あなたの意見を述べてください。

**ケース**：
```
ある企業では、従業員のPC操作ログをAIで分析し、
「生産性スコア」を算出するシステムを導入しました。
スコアの低い従業員には上司から面談が行われます。

【賛成派の意見】
・客観的な評価ができる
・サボりを防止できる
・生産性向上に寄与する

【反対派の意見】
・プライバシーの侵害
・精神的なストレスを与える
・創造性を阻害する
```

**分析の観点**：
1. ステークホルダー（誰が影響を受けるか）
2. 倫理原則（人間中心、公平性、透明性等）からの評価
3. 法的リスク
4. 代替案の検討
5. あなたの結論と理由

## 総合演習

### 総合演習：AI活用プロジェクト

**問題**：以下の状況設定で、AI活用プロジェクトを完遂してください。

**状況**：
```
あなたは、一般社団法人健全AI教育協会（HAIIA）から、
「地域の中小企業向けAI活用セミナー」の企画・運営を依頼されました。

対象：従業員50名以下の地域中小企業の経営者・管理職
目的：AI活用の基礎知識習得と、自社への適用イメージの獲得
形式：半日（4時間）のセミナー
参加予定：30名
予算：30万円（講師料、会場費、資料費込み）
```

**作成するもの**：
1. セミナー企画書
   - コンセプト・目的
   - ターゲットと期待効果
   - プログラム構成（タイムテーブル）
   - 必要なリソース・予算計画

2. 研修資料（主要パートの概要）
   - AI入門（10分相当）
   - 中小企業のAI活用事例（10分相当）
   - ワークショップ設計（15分相当）

3. 効果測定計画
   - 事前・事後アンケート項目
   - 成功の定義とKPI

4. リスクと対策
   - 想定されるリスク
   - 対応策

5. P-A-I-Cプロセスの記録
   - どのようなプロンプトを使ったか
   - 何回の反復を行ったか
   - どのように改善したか

**評価基準**：
- 企画の実現可能性と魅力
- AI活用の適切さ
- HAIIAの理念との整合性
- 成果物の品質
- P-A-I-Cプロセスの記録
