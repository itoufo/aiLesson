## 理念と教育方針

### HAIIAの5つの価値観

![HAIIAの5つの価値観](/docs/images/00_概要/00_haiia_values.jpeg)

### AI倫理5原則（OECD + IPA準拠）

1. **人間中心** - AIは人間の尊厳と幸福を最大化するための手段
2. **公平性** - バイアスを検証し多様な視点を尊重
3. **透明性** - プロンプト・出力・編集履歴を可視化
4. **安全性** - リスク管理と説明責任の明確化
5. **プライバシー** - 最小限取得・暗号化・期限付き保管

## AI時代に必要な4つの力

本カリキュラムでは、生成AI時代に共通して求められる基盤的コンピテンシーとして「4つの力」を軸に学習を設計しています。

![AI時代に必要な4つの力](/docs/images/00_概要/00_four_skills.jpeg)

### 4つの力の詳細

| 力 | 定義 | 主な学習活動 | 期待される効果 |
|----|------|------------|--------------|
| コミュニケーション力 | 人とAIの対話を設計・調整し、情報を構造化して合意を導く力 | プロンプト設計、意見統合、対話ログのメタ認知 | 多文化・多世代協働の円滑化 |
| 言語力 | 多モード情報を批判的に再構成し、分かりやすく発信する力 | 複数資料の批判的要約、多言語発信、ストーリーテリング | 誤情報への耐性向上、知の民主化 |
| セルフコーチング力 | 目標を適応的に更新し、行動を自己調整する力 | AIダッシュボードで自己モニタリング、リフレクション | 生涯学習の実装、自己効力感向上 |
| プロジェクトマネジメント力 | 目標・人・資源・リスクを統合管理し価値を創出する力 | WBS運用、リスク分析、成果物レビュー | 課題解決力、イノベーション創出 |

## レベル別評価システム（Lv1〜Lv5）

学習到達度を5段階で評価し、年齢や職務に依存しない「伸びしろ指標」として活用します。

| Lv | 段階 | コミュニケーション力 | 言語力 | セルフコーチング力 | PM力 |
|----|------|------------------|--------|-----------------|------|
| **Lv1** | 入門 | AI/相手発言を100字要約 | 語彙・文法誤りを抽出 | 1週間行動目標を列挙 | タスク優先度を並べ替え |
| **Lv2** | 基礎 | 確認質問を2回以上挿入 | 用途別リライト＋出典提示 | SMART目標＋週次入力 | WBSに担当と期限を反映 |
| **Lv3** | 実践 | 誤情報指摘＋修正プロンプト | 3資料統合で提案文書化 | データから課題分析→改善 | ガント更新＋リスク報告 |
| **Lv4** | 応用 | 異なる立場をファシリテート | ペルソナ別1000字提案 | ビジョン整合→ピア支援 | 品質・コミュニケーション計画 |
| **Lv5** | 専門 | 対話設計ガイドラインを策定 | 複合メディアで包括的発信 | 組織KPIと連動し行動変容支援 | AI内蔵PMOを設計・運営 |

### 3観点評価（K-T-A）

| 観点 | 内容 | 評価方法 |
|------|------|---------|
| **K（知識・技能）** | ツール操作・概念理解・手続き的スキル | 確認テスト、実技チェック |
| **T（思考・判断・表現）** | 批判的思考・論理構築・意思決定 | パフォーマンス課題、プレゼン |
| **A（学びに向かう力）** | メタ認知・倫理観・協働態度 | ポートフォリオ、リフレクション |

## 各段階の概要

### 第1段階：入門（基礎）

| 項目 | 内容 |
|------|------|
| テーマ | AI基礎と安全な活用 |
| 学習時間 | 6〜10時間 |
| 前提知識 | 不要（初心者歓迎） |
| 目標レベル | Lv1〜Lv2 |
| 修了条件 | 確認テスト合格（70%以上） |

**学習内容**
- AIリテラシー入門（AIとは何か）
- 生成AIの仕組みと原理（LLM、トークン化、確率的生成）
- AI倫理・安全性（著作権、プライバシー、バイアス）
- プロンプト基礎（GCOF構造：Goal-Condition-Output-Format）
- AI活用演習（要約・校正・言い換え）

**育成する力**
- コミュニケーション力（Lv1-2）：基本的なプロンプト設計
- 言語力（Lv1-2）：要約・校正スキル

---

### 第2段階：応用（実践）

| 項目 | 内容 |
|------|------|
| テーマ | 生成AIの活用スキルとビジネス応用 |
| 学習時間 | 20〜30時間 |
| 前提知識 | 第1段階修了 |
| 目標レベル | Lv2〜Lv3 |
| 修了条件 | 実践課題提出 + 確認テスト |

**学習内容**
- Chain-of-Thought（思考の連鎖）による問題解決
- 情報の整理・再構成・論理的表現（MECE、ピラミッド構造）
- プロジェクト管理へのAI活用（WBS、ガントチャート、KPI）
- 業務自動化ツール実践（Notion、Zapier等）
- ユニバーサルプロンプティング入門
- ビジネス課題解決ワークショップ

**育成する力**
- 言語力（Lv2-3）：情報統合・再構成
- プロジェクトマネジメント力（Lv2-3）：WBS・ガント運用

### 第3段階：発展（探究・プロジェクト）

| 項目 | 内容 |
|------|------|

| 学習時間 | 40〜60時間 |
| 前提知識 | 第2段階修了 |
| 目標レベル | Lv3〜Lv4 |
| 修了条件 | プロジェクト成果発表 + ピアレビュー |

**学習内容**

- セルフコーチングとリフレクション
- AI活用ポートフォリオ作成
- AI倫理週間（倫理課題討議）

**育成する力**
- セルフコーチング力（Lv3-4）：自己調整学習
- 4つの力の統合的活用

## P-A-I-Cサイクル

本カリキュラムの核となる学習フレームワークです。生成AIを"循環エンジン"として位置づけ、学習者の批判的思考と創造的思考を交互に活性化します。

![P-A-I-Cサイクル](/docs/images/00_概要/00_paic_cycle.jpeg)

| フェーズ | 主要行動 | 4つの力との関連 |
|---------|---------|---------------|
| **Prompt** | 問題・テーマをAIと協議し、探究問いを確定 | コミュニケーション力 |
| **Ask** | 資料検索・データ収集・仮説立案 | 言語力 |
| **Iterate** | プロトタイプ生成→評価→改善 | セルフコーチング力 |
| **Create** | 成果物を公開・共有 | プロジェクトマネジメント力 |

## ユニバーサルプロンプティング

「誰でも・どの文脈でも再利用しやすいプロンプト設計法」を学びます。

### 基本構造

```
【目的】What - 何を達成したいか
【変数】Who / When / Where - 誰が、いつ、どこで
【コンテキスト】Why - なぜそれが必要か（背景）
【制約条件】How - 形式・語数・トーン
【出力形式】フォーマット指定
```

### 温度設定の使い分け

| 温度 | 用途 | 特徴 |
|------|------|------|
| 低温（0.2-0.3） | 収束的タスク | 校正、要約、事実確認 |
| 中温（0.5-0.7） | バランス | 一般的な文書作成 |
| 高温（0.8-1.0） | 創造的タスク | アイデア出し、創作 |
