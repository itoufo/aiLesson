# 第5章：ビジネス課題解決

## 学習目標

この章を終えると、以下のことができるようになります：

- AIを活用したビジネス課題解決の全体プロセスを実践できる
- 課題の定義から解決策の提案まで一貫して取り組める
- 複数のAIスキルを組み合わせて活用できる
- 実務で使えるアウトプットを作成できる

## 5.2 ケーススタディ1：売上低下の分析

### 5.2.1 ケース概要

```
【状況】
あなたは中堅EC企業のマーケティング担当です。
直近3ヶ月で売上が前年比15%減少しています。
経営層から原因分析と対策の提案を求められました。

【データ】
- 月間売上：1月 8,000万円、2月 7,500万円、3月 7,200万円
- 前年同期：1月 9,200万円、2月 8,800万円、3月 8,500万円
- 訪問者数：前年比 -5%
- CVR：前年3.2% → 今年2.5%
- 客単価：前年8,000円 → 今年7,500円
- リピート率：前年40% → 今年32%
```

### 5.2.2 Step 1：課題の定義

```markdown
【AIへのプロンプト】

以下の状況について、課題を明確に定義してください。

状況：
EC企業で直近3ヶ月の売上が前年比15%減少
- 訪問者数：-5%
- CVR：3.2%→2.5%
- 客単価：8,000円→7,500円
- リピート率：40%→32%

以下の形式で整理してください：

1. 問題の定量化
   - 売上減少額：○○円
   - 各要因の影響度

2. 課題の構造化（イシューツリー形式）

3. 優先的に取り組むべき課題
```

### 5.2.3 Step 2：現状分析

```markdown
【AIへのプロンプト】

以下のデータを分析し、現状を整理してください。

データ：
[上記のデータを記載]

分析の観点：
1. 売上の構成要素分解（訪問者×CVR×客単価）
2. 各指標の前年比較とトレンド
3. 競合や市場環境の考慮点

出力形式：
- 数値分析の結果表
- 主要な発見事項（3-5点）
- グラフ化すべきデータの提案
```

### 5.2.4 Step 3：原因特定

```markdown
【AIへのプロンプト】

CVRと リピート率の低下について、原因を深掘りしてください。

5回の「なぜ」を繰り返し、根本原因を特定してください。

【CVR低下の原因分析】
なぜ1：CVRが低下した
なぜ2：（AIが推論）
なぜ3：（AIが推論）
なぜ4：（AIが推論）
なぜ5：（AIが推論）→根本原因

【リピート率低下の原因分析】
（同様に分析）

最後に、検証すべき仮説をリストアップしてください。
```

### 5.2.5 Step 4：解決策立案

```markdown
【AIへのプロンプト】

特定した原因に対する解決策を立案してください。

原因：
- CVR低下の根本原因：[特定した原因]
- リピート率低下の根本原因：[特定した原因]

各原因に対して：
1. 短期施策（1ヶ月以内に実行可能）を3つ
2. 中期施策（3ヶ月以内）を3つ
3. 長期施策（6ヶ月以上）を2つ

各施策について以下を記載：
- 施策内容
- 期待効果（定量）
- 必要リソース
- リスク・課題
```

### 5.2.6 Step 5：提案書作成

```markdown
【AIへのプロンプト】

経営層向けの提案書を作成してください。

含める内容：
- エグゼクティブサマリー
- 現状分析結果
- 原因と解決策
- 実行計画
- 期待効果
- 必要投資と ROI

形式：
- 10枚程度のスライド構成
- 各スライドのタイトルと主要メッセージ
- キーとなるグラフ・図表の提案
```

## 5.3 ケーススタディ2：業務効率化

### 5.3.1 ケース概要

```
【状況】
あなたは総務部の担当者です。
毎月の経費精算業務に多くの時間がかかっており、
効率化の提案を求められています。

【現状の業務フロー】
1. 社員が紙の申請書に記入（15分/件）
2. 領収書を添付して提出
3. 総務が内容確認・入力（10分/件）
4. 上長承認（紙で回覧、平均3日）
5. 経理が会計システムに入力（5分/件）
6. 月次で集計レポート作成（8時間/月）

【規模】
- 月間申請件数：500件
- 対象社員：200名
- 総務担当者：3名
```

### 5.3.2 問題解決のプロセス

```markdown
【Step 1: 課題の定義】
AIプロンプト：
「上記の業務フローについて、課題を洗い出し、
 影響度と緊急度でマトリクス化してください」

【Step 2: 現状分析】
AIプロンプト：
「現状の業務工数を計算し、ボトルネックを特定してください」

計算例：
- 社員の申請時間：500件 × 15分 = 125時間/月
- 総務の処理時間：500件 × 10分 = 83時間/月
- 経理の入力時間：500件 × 5分 = 42時間/月
- レポート作成：8時間/月
- 合計：258時間/月（約1.6人月）

【Step 3: 原因特定】
AIプロンプト：
「なぜこれほど時間がかかっているか、
 業務の各ステップで非効率が生じている原因を分析してください」

【Step 4: 解決策立案】
AIプロンプト：
「経費精算業務を自動化・効率化するソリューションを提案してください。
 クラウドサービス、ワークフローシステム、AI-OCR等を検討に含めてください」

【Step 5: 実行計画】
AIプロンプト：
「選定したソリューションの導入計画を策定してください。
 WBS、スケジュール、予算、期待効果を含めてください」
```

## 5.4 ケーススタディ3：新規事業企画

### 5.4.1 ケース概要

```
【状況】
あなたは事業企画部の担当者です。
「既存顧客基盤を活かした新規事業」の企画を求められています。

【会社概要】
- 業種：法人向けオフィス用品通販
- 顧客数：5,000社
- 年商：50億円
- 強み：確実な配送、豊富な品揃え、法人与信ノウハウ
```

### 5.4.2 新規事業企画プロセス

```markdown
【Step 1: 機会の探索】
AIプロンプト：
「法人向けオフィス用品通販会社が展開できる新規事業のアイデアを
 10個提案してください。既存の強みを活かせるものを優先してください」

【Step 2: アイデアの評価】
AIプロンプト：
「提案されたアイデアを以下の基準で評価してください：
 - 市場規模
 - 既存アセット活用度
 - 参入障壁
 - 収益性
 - リスク
 各項目5点満点で採点し、総合評価してください」

【Step 3: 有望アイデアの深掘り】
AIプロンプト：
「上位3つのアイデアについて、以下を詳細に分析してください：
 - ターゲット顧客像
 - 価値提案（バリュープロポジション）
 - 収益モデル
 - 競合状況
 - 成功のための重要成功要因」

【Step 4: ビジネスモデル設計】
AIプロンプト：
「最も有望なアイデアについて、ビジネスモデルキャンバスを作成してください」

【Step 5: 事業計画書作成】
AIプロンプト：
「新規事業の提案書を作成してください。
 - 市場機会
 - 事業コンセプト
 - ビジネスモデル
 - 競争優位性
 - 財務計画（3年）
 - マイルストーン
 - リスクと対策」
```

## 5.5 統合演習

### 5.5.1 総合演習問題

```
【演習課題】

あなたは中堅製造業（従業員300名）のDX推進担当に任命されました。
経営層から「3年以内にDXで競争力を強化せよ」という指示を受けています。

与えられた情報：
- 主力製品：工業用部品
- 売上：100億円（横ばい）
- 課題：納期遅延、品質クレーム増加、人手不足
- IT環境：基幹システムは20年前のもの、Excel多用
- 予算：DX投資として3年で3億円

以下のステップで提案書を作成してください：

1. 現状分析と課題の整理
2. DX施策の優先順位付け
3. 3年間のロードマップ
4. 投資対効果（ROI）試算
5. 経営層向け提案書
```

### 5.5.2 演習の進め方

```
【推奨プロセス】

1. 情報整理（30分）
   - 与えられた情報を構造化
   - 不足情報の特定と仮定の設定

2. 課題分析（30分）
   - SWOT分析
   - 課題の優先順位付け

3. 施策立案（45分）
   - DX施策のアイデア出し
   - 評価と選定

4. 計画策定（30分）
   - WBS作成
   - スケジュール策定
   - 予算配分

5. 提案書作成（45分）
   - ストーリーライン設計
   - 各セクションの執筆

合計：3時間
```

### 5.5.3 評価基準

```
【提案書の評価基準】

1. 論理性（25点）
   - 課題分析が論理的か
   - 施策と課題が紐づいているか
   - 根拠が明確か

2. 実現可能性（25点）
   - 予算内に収まっているか
   - スケジュールが現実的か
   - リソースの考慮があるか

3. 効果の明確さ（25点）
   - KPIが設定されているか
   - ROIが試算されているか
   - 定性効果も言及されているか

4. プレゼンテーション（25点）
   - 構成がわかりやすいか
   - キーメッセージが明確か
   - 視覚的に見やすいか
```

## 章末まとめ

### 本章のポイント

1. ビジネス課題解決は**5ステップ**（定義→分析→原因特定→解決策→計画）

3. 複数のスキルを**組み合わせて**実務課題に取り組む
