# 第1章：AIリテラシー入門

## 学習目標

この章を終えると、以下のことができるようになります：

- AIの基本的な定義と種類を説明できる
- AIの「できること」と「できないこと」を区別できる
- AIの発展の歴史と現在の位置づけを理解できる
- 日常生活やビジネスにおけるAI活用事例を挙げられる

---

## 1.1 AIとは何か

### 1.1.1 AIの定義

**AI（人工知能）** とは、人間の知的活動（学習、推論、認識、判断など）をコンピュータで実現する技術の総称です。

![AI技術の階層構造](/docs/images/01_入門/01_ai_hierarchy.jpeg)

### 1.1.2 AIの種類

| 種類 | 説明 | 例 |
|------|------|-----|
| **特化型AI（弱いAI）** | 特定のタスクに特化 | 画像認識、音声認識、チャットボット |
| **汎用AI（強いAI）** | 人間と同等の知能（未実現） | SF映画のAI |
| **生成AI** | 新しいコンテンツを生成 | ChatGPT、Claude、Midjourney |

### 1.1.3 現在のAIの特徴

現在実用化されているAIは**特化型AI**であり、以下の特徴があります：

- 大量のデータから**パターンを学習**する
- 学習した範囲内で**高精度な予測・生成**ができる
- 学習データに含まれない状況には**対応が難しい**
- **意識や感情は持たない**（持っているように見えることはある）

## 1.3 AIの「できること」と「できないこと」

### 1.3.1 AIが得意なこと

| 分野 | 具体例 |
|------|--------|
| **パターン認識** | 画像認識、音声認識、異常検知 |
| **大量データ処理** | データ分析、検索、要約 |
| **反復作業の自動化** | 定型文作成、データ入力、翻訳 |
| **予測・推薦** | 需要予測、レコメンデーション |
| **コンテンツ生成** | 文章作成、画像生成、コード生成 |

### 1.3.2 AIが苦手なこと

| 分野 | 理由 |
|------|------|
| **創造的な発想** | 学習データの範囲を超えた真の創造は困難 |
| **常識的判断** | 人間が当然と思う背景知識の欠如 |
| **倫理的判断** | 価値観や文脈に基づく判断が難しい |
| **責任を持つこと** | 法的・道義的責任は人間が負う |
| **最新情報の把握** | 学習データの時点で知識が固定 |

## 1.4 生成AIの基本

### 1.4.1 生成AIとは

**生成AI（Generative AI）** は、学習したデータのパターンを基に、新しいコンテンツ（文章、画像、音声、コードなど）を生成するAIです。

### 1.4.2 代表的な生成AIサービス

| サービス | 提供元 | 主な機能 |
|----------|--------|----------|
| ChatGPT | OpenAI | 対話型文章生成 |
| Claude | Anthropic | 対話型文章生成・分析 |
| Gemini | Google | マルチモーダル対話 |
| Midjourney | Midjourney | 画像生成 |
| GitHub Copilot | GitHub/OpenAI | コード生成 |

### 1.4.3 大規模言語モデル（LLM）の仕組み

![LLMの仕組み](/docs/images/01_入門/01_llm_mechanism.jpeg)

**重要**: LLMは「理解」しているように見えますが、実際は**統計的なパターンマッチング**を行っています。

## 1.5 AIの活用事例

### 1.5.1 日常生活での活用

- **スマートフォン**: 音声アシスタント、顔認証、写真の自動整理
- **ショッピング**: 商品レコメンデーション、価格予測
- **エンターテインメント**: 動画・音楽のレコメンド、ゲームのNPC
- **移動**: カーナビの渋滞予測、配車アプリの最適化

### 1.5.2 ビジネスでの活用

| 業界 | 活用例 |
|------|--------|
| **製造業** | 品質検査の自動化、予知保全、生産最適化 |
| **金融** | 不正検知、与信審査、アルゴリズム取引 |
| **医療** | 画像診断支援、創薬、患者トリアージ |
| **小売** | 需要予測、在庫最適化、パーソナライズ広告 |
| **サービス** | チャットボット、文書作成支援、翻訳 |

## 確認クイズ

### Q1. AIの定義
次のうち、AIの説明として最も適切なものはどれですか？

- [ ] A) 人間の脳を完全に再現したコンピュータ
- [ ] B) 人間の知的活動をコンピュータで実現する技術
- [ ] C) 自我を持ったコンピュータプログラム
- [ ] D) インターネットに接続されたすべてのコンピュータ

### Q2. AIの得意・不得意
次のうち、現在のAIが**苦手**とすることはどれですか？

- [ ] A) 大量のデータからパターンを見つける
- [ ] B) 画像の中から特定の物体を認識する
- [ ] C) 学習データにない状況での倫理的判断

- [ ] C) 学習データのパターンを基に確率的に次の単語を予測している

## 章末まとめ

### 本章のポイント

1. **AIは人間の知的活動を再現する技術**であり、現在実用化されているのは特化型AI
2. **AIには得意・不得意がある** - パターン認識は得意、創造的・倫理的判断は苦手
3. **生成AIは統計的パターンマッチング**で動作し、「理解」しているわけではない
4. **AIは人間を拡張する道具**であり、最終判断は人間が行う
